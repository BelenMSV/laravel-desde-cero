# Módulo 5: Formularios y Validación

Nuestra aplicación ya sabe mostrar datos de la base de datos, pero ahora necesitamos que los usuarios puedan enviar su propia información. En este módulo aprenderás a manejar formularios de forma segura, validar que los datos sean correctos y devolver una excelente experiencia de usuario cuando cometan errores.

## 📖 Temas del Módulo

1. **[Tema 1: Formularios y Seguridad CSRF](./01-formularios-y-csrf/README.md)** Aprende a crear formularios en Blade y descubre por qué la directiva `@csrf` es tu mejor escudo contra ataques cibernéticos.

2. **[Tema 2: Spoofing de Métodos (PUT, PATCH y DELETE)](./02-spoofing-de-metodos/README.md)** Supera las limitaciones del HTML tradicional. Descubre el truco de Laravel para actualizar y borrar registros usando `@method`.

3. **[Tema 3: Validación Básica](./03-validacion-basica/README.md)** Nunca confíes en el usuario. Aprende a usar `$request->validate()` para asegurar que te envíen emails reales, contraseñas seguras y campos obligatorios.

4. **[Tema 4: Feedback de Errores y la función old()](./04-errores-y-old/README.md)** Mejora la experiencia de usuario (UX). Muestra mensajes de error en rojo y evita que el usuario tenga que rellenar todo el formulario de nuevo si se equivoca.

5. **[Tema 5: Form Requests (El Estándar Profesional)](./05-form-requests/README.md)** Limpia tus controladores. Aprende a mover toda la lógica de validación a un archivo independiente para mantener tu código impecable.

6. **[Tema 6: Subida de Archivos e Imágenes](./06-subida-de-archivos/README.md)** Añade la funcionalidad que todo proyecto necesita. Aprende a recibir fotos, guardarlas en el disco público y mostrarlas en tus vistas.
