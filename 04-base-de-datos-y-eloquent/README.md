# Módulo 4: Base de Datos y Eloquent ORM

Este es el módulo más importante del curso. Aquí aprenderás cómo Laravel interactúa con el motor de base de datos para almacenar y recuperar información. Descubrirás la potencia de **Eloquent**, el ORM de Laravel, que te permite realizar consultas complejas usando código PHP elegante en lugar de SQL puro.

## 📖 Temas del Módulo

1. **[Tema 1: Configuración y Conexión](./01-configuracion-y-conexion/README.md)** Prepara tu entorno. Aprende a configurar el archivo `.env` y a elegir entre SQLite (ideal para aprender) o MySQL.

2. **[Tema 2: Migraciones](./02-migraciones/README.md)** El "Git" de las bases de datos. Aprende a crear y modificar tablas mediante código para que tu equipo siempre tenga la misma estructura.

3. **[Tema 3: Modelos y Eloquent ORM](./03-modelos-y-eloquent/README.md)** Entiende la pieza "M" del patrón MVC. Cómo crear modelos que representen tus tablas y las reglas de oro de los nombres en Laravel.

4. **[Tema 5: Laravel Tinker](./05-laravel-tinker/README.md)** La consola mágica. Aprende a interactuar con tu base de datos en tiempo real desde la terminal sin escribir una sola línea de HTML.

5. **[Tema 4: Factories y Seeders](./04-factories-y-seeders/README.md)** ¡No más bases de datos vacías! Aprende a generar miles de datos de prueba (usuarios, posts, comentarios) en segundos.

6. **[Tema 6: Consultas Básicas (CRUD con Eloquent)](./06-consultas-basicas/README.md)** Domina los métodos `all()`, `find()`, `where()`, `create()` y `update()`. Incluye la explicación vital sobre `$fillable`.

7. **[Tema 7: Introducción a las Relaciones](./07-relaciones/README.md)** El siguiente nivel. Aprende cómo conectar tablas entre sí (ej. un autor tiene muchos posts) usando `hasMany` y `belongsTo`.
