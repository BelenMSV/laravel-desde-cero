# Módulo 3: Controladores y Arquitectura MVC

En este módulo daremos el salto a la programación profesional. Aprenderás a organizar tu código siguiendo el patrón de diseño más utilizado en la web y a separar la lógica de negocio de tus rutas para crear aplicaciones escalables y mantenibles.

## 📖 Temas del Módulo

1. **[Tema 1: El Patrón MVC](./01-el-patron-mvc/README.md)** Entiende la teoría detrás del Modelo-Vista-Controlador y cómo Laravel coordina estas tres capas para procesar las peticiones.

2. **[Tema 2: Creación y Rutas de Controladores](./02-creacion-y-rutas/README.md)** Aprende a generar controladores con Artisan y a delegar la lógica de tus rutas a métodos específicos dentro de clases.

3. **[Tema 3: El Objeto Request](./03-el-objeto-request/README.md)** Domina la captura de datos. Aprende a inyectar la clase `Request` para obtener información de formularios, cabeceras y parámetros de URL de forma segura.

4. **[Tema 4: Controladores de Recursos](./04-controladores-de-recursos/README.md)** El estándar de Laravel para CRUDS. Descubre cómo generar automáticamente los 7 métodos estándar para gestionar un recurso con una sola línea de código.

5. **[Tema 5: Controladores Invocables](./05-controladores-invocables/README.md)** Aprende a simplificar tu arquitectura creando controladores de acción única para procesos específicos y aislados.
