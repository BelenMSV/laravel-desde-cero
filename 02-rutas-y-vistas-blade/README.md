# Módulo 2: Rutas y Vistas Blade

En este módulo, le dimos vida a nuestra aplicación. Aprendimos cómo Laravel maneja las peticiones de los usuarios a través de las URLs y cómo mostrar interfaces gráficas dinámicas utilizando el motor de plantillas Blade.

## 📖 Temas del Módulo

1. **[Tema 1: Rutas Básicas y Parámetros](./01-rutas-basicas-y-parametros/README.md)** Aprende a definir las URLs de tu aplicación en el archivo `web.php`, usar los métodos HTTP (como `GET`) y capturar datos dinámicos pasando variables a través de la dirección web.

2. **[Tema 2: Introducción a Blade](./02-introduccion-a-blade/README.md)** Descubre cómo separar tu lógica del HTML usando el motor de plantillas de Laravel. Aprenderás a enviar variables a las vistas y a utilizar directivas clave como `@if` y `@foreach`.

3. **[Tema 3: Layouts y Componentes](./03-layouts-y-componentes/README.md)** Domina el principio DRY (*Don't Repeat Yourself*). Aprenderás a crear una estructura base o plantilla maestra (`<x-layout>`) para tu sitio web y a inyectar contenido usando la variable `$slot`.

4. **[Tema 4: Rutas con Nombre](./04-rutas-con-nombre/README.md)** Una buena práctica esencial. Descubre cómo asignar alias a tus rutas (`->name()`) y generar enlaces dinámicos en Blade con el helper `route()` para que tu código sea inmune a futuros cambios de URL.

5. **[Tema 5: Archivos Estáticos](./05-archivos-estaticos-y-assets/README.md)** Entiende la función de la carpeta `public/` y aprende a vincular de forma segura tus hojas de estilo (CSS), scripts (JS) y logotipos utilizando la función mágica `asset()`.
