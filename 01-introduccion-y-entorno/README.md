# Módulo 1: Introducción y Entorno

En este primer bloque del curso, sentaremos las bases teóricas y técnicas necesarias para trabajar con Laravel. Antes de escribir nuestra primera línea de código, es fundamental entender la herramienta que estamos usando y preparar un entorno de desarrollo profesional.

## 📖 Temas del Módulo

1. **[Tema 1: ¿Qué es Laravel?](./01-que-es-laravel/README.md)** Una introducción al framework, sus ventajas y por qué es el favorito de la comunidad PHP. Conoce a nivel teórico conceptos clave como el patrón MVC y el ORM Eloquent.

2. **[Tema 2: Instalación y Requisitos](./02-instalacion-y-requisitos/README.md)** Guía paso a paso para preparar tu entorno local. Aprende a instalar herramientas esenciales como PHP y Composer (usando Herd o Laragon), crear tu primer proyecto con `create-project` y levantar el servidor de desarrollo.

3. **[Tema 3: Estructura de Archivos](./03-estructura-de-archivos/README.md)** Un recorrido para perderle el miedo a las carpetas del proyecto. Descubre dónde vive tu código (`app/`, `routes/`, `resources/`, `public/`) y la importancia crítica de ocultar el archivo de configuración `.env`.

4. **[Tema 4: Configuración del Editor](./04-configuracion-del-editor/README.md)** Optimización de Visual Studio Code (VS Code). Conoce las extensiones imprescindibles (como PHP Intelephense y Laravel Blade Snippets) para tener autocompletado y programar más rápido.

5. **[Tema 5: Introducción a Artisan](./05-introduccion-a-artisan/README.md)** Primer contacto con la potente interfaz de línea de comandos (CLI) de Laravel. Aprende a usar `php artisan list` y el comando mágico `make:` para automatizar tareas y generar archivos rápidamente.
