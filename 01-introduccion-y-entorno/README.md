# Módulo 1: Introducción y Entorno

En este primer bloque del curso, sentaremos las bases teóricas y técnicas necesarias para trabajar con Laravel. Antes de escribir nuestra primera línea de código, es fundamental entender la herramienta que estamos usando y preparar un entorno de desarrollo profesional.

## 📖 Temas del Módulo

A continuación, encontrarás los temas detallados de este módulo:

1.  **[Tema 1: ¿Qué es Laravel?](./01-que-es-laravel/README.md)** Una introducción al framework, sus ventajas y por qué es el favorito de la comunidad PHP.
2.  **[Tema 2: Instalación y Requisitos](./02-instalacion-y-requisitos/README.md)** Guía paso a paso para instalar PHP, Composer y crear tu primer proyecto.
3.  **[Tema 3: Estructura de Archivos](./03-estructura-de-archivos/README.md)** Un recorrido por las carpetas del framework para que sepas dónde encontrar cada cosa.
4.  **[Tema 4: Configuración del Editor](./04-configuracion-del-editor/README.md)** Optimización de Visual Studio Code con las mejores extensiones para Laravel.
5.  **[Tema 5: Introducción a Artisan](./05-introduccion-a-artisan/README.md)** Primer contacto con la potente interfaz de línea de comandos de Laravel.
