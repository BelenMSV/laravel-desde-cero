# Tema 1: ¿Qué es Laravel?

¡Bienvenido al primer tema de nuestro curso! Antes de ensuciarnos las manos escribiendo código, es fundamental entender qué herramienta vamos a utilizar y por qué se ha vuelto tan popular en el mundo del desarrollo web.

## ¿Qué es exactamente Laravel?

**Laravel** es un *framework* de código abierto para el lenguaje de programación **PHP**. Fue creado por Taylor Otwell en 2011 con un objetivo principal: hacer que el desarrollo web fuera más elegante, rápido y agradable para el programador.

Pero, ¿qué es un *framework*? 
Piensa en un framework como los cimientos y pilares de una casa prefabricada. En lugar de tener que mezclar el cemento y poner ladrillo por ladrillo desde cero cada vez que inicias un proyecto nuevo, el framework ya te da toda una base estructurada, segura y lista para funcionar. Tú solo te encargas de la distribución de las habitaciones y la decoración (la lógica de tu aplicación).

## ¿Por qué elegir Laravel? (Características principales)

Laravel destaca sobre otras alternativas en PHP por varias razones clave:

* **Sintaxis elegante y expresiva:** El código en Laravel está diseñado para ser fácil de leer y escribir. Se siente casi como lenguaje natural, lo que hace que programar sea menos tedioso.
* **Arquitectura MVC:** Utiliza el patrón de diseño Modelo-Vista-Controlador (MVC). Esto te obliga a mantener el código organizado, separando la lógica de la base de datos de la interfaz gráfica que ve el usuario.
* **Eloquent ORM:** Es el "mago" de las bases de datos en Laravel. En lugar de escribir complejas consultas SQL a mano, interactuarás con tus tablas usando código PHP sencillo y orientado a objetos.
* **Blade Templating:** Un motor de plantillas súper potente que permite inyectar variables de PHP directamente en tu HTML de forma limpia y segura.
* **Comunidad gigante y ecosistema:** Al ser el framework de PHP más popular del mundo, si alguna vez te atascas con un error, es casi seguro que alguien más ya lo resolvió en foros, GitHub o StackOverflow.

## ¿Qué conocimientos previos necesitas?

Este curso está diseñado para ir paso a paso, pero para sacar el máximo provecho y no frustrarte en el camino, es recomendable tener nociones básicas de:

1.  **HTML básico:** Para entender la estructura de las vistas que le mostraremos al usuario.
2.  **Fundamentos de PHP:** Variables, bucles (`for`, `foreach`), arreglos (`arrays`) y funciones.
3.  **Programación Orientada a Objetos (POO):** Conceptos básicos como qué es una Clase, un Objeto, un Método y una Propiedad, ya que Laravel está construido 100% bajo este paradigma.
